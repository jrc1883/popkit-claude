## PDF Styling Guidelines

### Typography
- **Headers**: Bold, larger font size
- **Body**: Professional serif or sans-serif (11-12pt)
- **Code**: Monospace font with background shading
- **Tables**: Clean borders, alternating row colors

### Layout
- **Margins**: 1 inch minimum
- **Page numbers**: Bottom center or right
- **Headers/Footers**: Include document title and date
- **Section breaks**: Clear visual separation

### Visual Elements
- Charts and graphs should be high-resolution
- Screenshots should be clearly annotated
- Color scheme should be consistent and print-friendly

## When to Use

- Formal stakeholder presentations
- Compliance documentation
- Audit reports
- Security assessments
- Project status reports for management
